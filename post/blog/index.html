#二、搭建hugo博客

###1.安装hugo

a.去github.com/gohugoio/hugo/看看 b.直接命令行下载安装 sudo apt-get install hugo 判断安装是否完成：hugo version

###2.用hugo来安装博客 hugo new site HugoBlog（在当前目录下生成一个HugoBlog文件夹）

###3.下载并设置主题 去hugo主题官网,按照里面的命令进行安装

###4.在本地启动博客 hugo server -t hallo --buildDrafts

###5.写一篇博文 hugo new post/blog.md
